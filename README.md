## 📋 **PREREQUISITES**

- **Node.js** v18 or higher
- **WhatsApp Account** for bot number
- **API Keys** (OpenAI/Gemini for AI features)
- **Git** installed on your system
- **Terminal/Command Line** access

## ⚙️ **CONFIGURATION**

1. **Rename `.env.example` to `.env`**
2. **Add your API keys:**
```env
